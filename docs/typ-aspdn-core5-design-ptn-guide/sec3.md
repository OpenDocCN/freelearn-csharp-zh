# 第三部分：组件级别的设计

本节重点介绍组件设计，我们将研究如何精心设计一个软件片段以实现特定目标。我们通过探索一些结构化的 GoF 模式来帮助设计可靠的数据结构和组件。通过将逻辑封装在更小的单元中，它们也有助于简化代码的复杂性。

我们继续介绍两种行为模式，它们有助于管理共享逻辑或简化管理复杂逻辑所需的工作。在本节的结尾，我们将探讨如何在组件之间传输有关操作错误和成功的结构化信息。
