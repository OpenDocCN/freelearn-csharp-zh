# 第二部分：设计解决方案架构

在本节中，我们将讨论解决方案架构的关键原则，并深入探讨最常用的 UML 图及其具体示例。然后，我们将了解使用 UML 创建和设计架构的过程。

之后，我们将探索关键架构模式和如何为我们的解决方案选择正确的模式。此外，我们将探讨解决方案架构的设计和运行时质量属性。在本节的后面部分，我们将深入研究安全考虑事项以及如何保护我们的 ASP.NET 解决方案，然后我们将讨论主要的测试类型以及在此方面的最佳实践。

+   *第八章*, *在解决方案架构中进行测试*
