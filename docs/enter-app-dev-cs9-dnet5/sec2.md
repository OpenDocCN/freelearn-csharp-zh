# 第二部分：交叉问题

目前我们有一个企业应用的框架结构。在用业务和技术功能填充这个框架时，我们会遇到许多将在应用的各个层中使用的代码和结构。这些有时被称为交叉问题。这些交叉问题包括线程、集合、日志、缓存、配置、网络和依赖注入。在这一部分，我们将从的角度快速回顾这些基本原理.NET 5 和企业应用。在每一章中，我们将学习这些交叉关注点如何构成企业应用的构建块，以及它们如何与用户界面和服务层集成。
