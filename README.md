## 下载

### Docker

```
docker pull apachecn0/apachecn-csharp-zh
docker run -tid -p <port>:80 apachecn0/apachecn-csharp-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-csharp-zh
apachecn-csharp-zh <port>
# 访问 http://localhost:{port} 查看文档
```
